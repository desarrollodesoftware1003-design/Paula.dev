ReservaFácil

Aplicación web para la gestión de reservas de vuelos y hoteles, diseñada para facilitar la administración de viajes mediante una experiencia sencilla, organizada, segura y fácil de usar.

Java Spring Boot Spring Security Hibernate / JPA MySQL Thymeleaf BCrypt JavaMailSender

Características principales

🔐 Autenticación segura 👤 Registro de usuarios ✈️ Reserva de vuelos 🏨 Reserva de hoteles 🧭 Panel de usuario intuitivo 📧 Recuperación de contraseña 👥 Gestión de usuarios 📋 Gestión de reservas 🚪 Cierre de sesión seguro

Este proyecto fue desarrollado aplicando principios de desarrollo backend, seguridad, persistencia de datos y una arquitectura organizada y escalable.
